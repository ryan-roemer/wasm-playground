===============

## Installation

Get Emscripten:

```sh
$ curl -Os https://s3.amazonaws.com/mozilla-games/emscripten/releases/emsdk-portable.tar.gz
$ tar zxvf emsdk-portable.tar.gz
$ cd emsdk-portable
$ ./emsdk update
$ cd ..
```

From there, and on any new session, "activate" the build environment:

```sh
$ yarn run activate
```
